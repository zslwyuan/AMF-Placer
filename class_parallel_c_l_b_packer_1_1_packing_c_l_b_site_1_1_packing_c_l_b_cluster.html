bool ParallelCLBPacker::PackingCLBSite::PackingCLBCluster::addFF ( DesignInfo::DesignCell *  curFF,
int  enforceHalfCLB = -1,
bool  enforceMainFFSlot = false 
)

try to add a given FF into this cluster

Parameters
curFFa given FF
enforceHalfCLB(default -1/no limit) limit the candidate half CLB id for this FF
enforceMainFFSlot(default false) constaint that the FF can be only placed in the main half CLB (the one connected to LUT6 output pins in the CLB site). There are two types of half CLBs, we call those connected to LUT6 output pins "main half CLB slots" and those connected to LUT5 output pins "secondary half CLB slots". Some FFs (especially the virtual ones) should be mapped to the "main" ones.
Returns
true if this attempt is successful
false if this attempt FAILED.
