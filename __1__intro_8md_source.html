3 **Motivations**
4 
5 1. Just reinvent the wheel for fun, try to build a complete flow and implement some state-of-art techniques in the latest paper.
6 2. Resolve some existing constraints in some previous works and consider more practical situations, like FPGA mixed-size placement with a series of optimization.
7 3. A beginner-friendly placement framework with clear hierarchy and detailed Doxygen-based documentation. We hope that it can lower the overhead for people who are also interested in this research area.
8 4. Currently, this framework is under development and it is still far from our goals and the practical demands, but we are happy to share our progress in this GitHub repository. If you have any questions/problems/suggestions, please contact feel free to contact us (tliang AT connect DOT ust DOT hk)
9 
10 
11 **Features**
12 
13 1. supports placeemnt with a large number of mixed-size macros with shape constraints in practical FPGA applications.
14 2. a set of optional optimization techniques to improve mixed-size FPGA placement QoR
15 3. parallelize the implementation of each stage of placement based on multi-threading
16 4. modularized function implementation for easier further development
17 5. flexible and extensible JSON-based placement configuration
18 6. a set of pre-implementation benchmarks from latest practical FPGA applications

26 **Project Hiearchy**
27 
28 Below is a hiearchy tree of this project. As for the details, please refer to the class information and collaboration diagram in the Doxygen documentation and trace the implementation, e.g., AMFPlacer, GlobalPlacer, and PlacementInfo.
29 ```
30 ├── benchmarks // benchmark information
31 │ ├── analysisScripts // experimental result analysis Python scripts
32 │ ├── testConfig // some test settings of placer
33 │ ├── VCU108 // information of design and device for VCU108
34 │ │ ├── compatibleTable // mapping information between design and device
35 │ │ ├── design // design information
36 │ │ └── device // device information
37 │ └── vivadoScripts // some Vivado Tcl scripts to extract design/device information
38 ├── build // potential target directory for built output
39 ├── doc // documentation-related materials
40 └── src // source code of the project
41  ├── app // application (e.g., AMFPlacer)
42  │ └── AMFPlacer // A High-Performance Analytical Mixed-size Placer for FPGA
43  └── lib // libraries for application implementation
44  ├── 3rdParty // third-party libraries
45  ├── HiFPlacer // our placement function modules
46  │ ├── designInfo
47  │ ├── deviceInfo
48  │ ├── placement
49  │ │ ├── globalPlacement
50  │ │ ├── legalization
51  │ │ ├── packing
52  │ │ ├── placementInfo
53  │ │ └── placementTiming
54  │ └── problemSolvers
