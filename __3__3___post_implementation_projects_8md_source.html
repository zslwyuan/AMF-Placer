13 Users can directly open the .xpr file with Vivado. There might be some warnings because your Vivado version is not matched with the IP cores in the design or some IP core instances cannot be found in your system. Please ignore them and do not update the ip core or re-synthesis because these operations will change the netlist. Besides, if users use the benchmarks in their works, please cite the papers of the related designs and comply with their open-source licence conditions according to (@subpage _3_1_BenchmarksDetails).
